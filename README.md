# User Management App

Simple web app to manage users: add, delete, check age, calculate BMI, and list all users.

## Features
- Add new users with name, birth year, height, and weight
- Check user age
- Calculate BMI
- Delete users
- Display all users
- Check if user exists
- Check if user is adult

## How to use
1. Open `index.html` in your browser
2. Fill in user data and click the buttons

- HTML
- CSS
- JavaScript
